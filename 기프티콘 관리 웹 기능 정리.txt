기프티콘 관리 웹 기능 정리

1. 클라우드의 사진들 중에 기프티콘 사진(바코드 사진)을 분류한다.(데이터 베이스)
- 만약 클라우드가 어렵다면, 사용자가 직접 사진을 올린다.
- 환룡선배한테 구현이되는지 한 번 물어볼 것

2. 사진 속에 나와있는 정보를 긁어온다.(유효기간, 상품명, 판매기업, 금액 등)
ex) [CU]츄파츕스 2023-04-07
유효기간 - 2023-04-07 또는 2023.04.07 또는 2023년 4월 7일
상품명 - ?
판매기업 - []안의 정보 또는 기업명을 다 추가해서 매치가 되면 기업명
금액 - 금액이 나와있으면 "원"을 기준으로 띄어쓰기/엔터 전까지 글자
10000원 또는 만원 또는 1만원

- 사실 유효기간만 긁어와도 선방.
- 사용자가 직접 수정할 수 있는 기능 있으면 좋음.

3. 유효기간이 임박했을 때 알림 기능
- 웹에서 알림이 어렵기 때문에, 임박한 기프티콘은 상단에 따로 분류

4. 사용완료한 기프티콘은 사용자가 사용완료 버튼 누르면 삭제 

5. 사용기한이 만료된 기프티콘은 따로 분류


추가 기능

1. 사용자가 직접 사진과 유효기간 등 정보를 입력할 수 있다.
(인식 못하는 사진의 경우)

2. 커뮤니티를 만들어서 같은 금액대의 다른 기프티콘을 가진 사용자끼리 교환할 수 있다.
- 에타 형식

3. 같은 금액의 다른 상품 추천
ex) 만약 내가 라떼 쿠폰이있는데, 라떼가 싫어. 그러면 같은 금액대의 에이드를 추천.
ex) 만약 내가 치즈케이크 쿠폰이 있는데, 싫어. 그러면 아메리카노 + 라떼로 추천.
- 금액이 같거나 조금 넘는 상품으로 추천(+5000원까지)
